## 🛠️ Folder Structure Guide

Each folder is organized to keep the learning process modular and easy to review:

* **Notes:** Conceptual explanations, logic breakdowns, and pseudocode.
* **Code:** Well-commented solutions (primarily `.ipynb` or `.py`).
* **Complexity:** Detailed Time ($O(N)$) and Space complexity analysis for every problem.
